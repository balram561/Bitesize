## Technologies Used :
* Frontend : **XML**
* Backend : **JAVA**
* Database : **Firebase** & **SQLite**
* Android Studio
* Android SDK tools

## Features :
* Account Sign Up/ Login using personal details or **Social Login**
* Menu Page
* Food detail page
* Cart - Add / Remove
* Payment Gateway
* SMS & Email notification
* User Dashboard
* Live location tracking
